# Data Science with a Heart 
**Team Lovelace - CE101 Project Report**

## Project Overview
"Data Science with a Heart" is a Clinical Decision Support System (CDSS) designed to assist healthcare professionals in predicting heart disease risk. Using a machine learning pipeline grounded in the **UCI Cleveland Heart Disease Dataset**, the system analyzes 14 clinical variables to estimate the likelihood of cardiovascular issues.

## Technical Results
- **Model:** Logistic Regression (Optimized via GridSearchCV)
- **Validation Accuracy:** 82.85%
- **Data Architecture:** 80/20 Train-Test Split
- **Frameworks:** Scikit-learn, Pandas, Matplotlib, Seaborn

## Repository Structure
- `Final code`: The primary production notebook containing the optimized model.
- `data cleaning`: Pre-processing scripts for sanitizing the UCI dataset.
- `up_till_week_X`: Developmental iterations showing the project's evolution.
